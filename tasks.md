Tarefas  
Use este checklist para ajudar a organizar a sua entrega

☐ Desenvolver a rota `POST /leads` para criar um novo lead  
☐ Desenvolver a rota `GET /leads` para listar todos os leads  
☐ Implementar a funcionalidade de busca por `name` e `email` na rota `GET /leads`  
☐ Desenvolver a rota `PUT /leads/:id` para atualizar um lead  
☐ Adicionar a validação de existência do `id` na rota `PUT /leads/:id`  
☐ Desenvolver a rota `DELETE /leads/:id` para remover um lead  
☐ Adicionar a validação de existência do `id` na rota `DELETE /leads/:id`  
☐ Desenvolver a rota `PATCH /leads/:id/contact` para marcar um lead como contatado/não contatado  
☐ Adicionar a validação de existência do `id` na rota `PATCH /leads/:id/contact`  
☐ Criar um script separado para a importação de leads  
☐ Utilizar a biblioteca `csv-parse` para ler o arquivo CSV de leads  
☐ Implementar a lógica de enviar requisição para `POST /leads` para cada linha do CSV no script de importação
